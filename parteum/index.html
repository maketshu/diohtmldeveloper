Ordenada ou não ordenada

(Voltar)

Para adicionar uma lista ordenada, ou sejam que possui números como indice utilize o comando < ol >.

No caso, este modelo possui uma lista não ordenada, representado por ponto - ali na região do indice, para listar a quantidade de "filhas" necessário adicionar a tag < l i > .

Edição pode ser realizada atraver do CSS.

Hiperlink e afins

(Voltar)

A tag < a > é associada a links: < a href="link desejado ou referencia da page utilizando ID + #" > Nome que deseja exibir < /a >

h1 à h6 são resposáveis pelos títulos da page
